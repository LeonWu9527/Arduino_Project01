# 簡易接線圖

以下是簡易的接線圖，幫助您快速了解各元件與 Arduino 的連接方式：

```
Arduino Uno R3
+-------------------+
| [A4] --- LCD1602 SDA  |
| [A5] --- LCD1602 SCL  |
| [10] --- HC-SR04 Trig |
| [11] --- HC-SR04 Echo |
| [2-5] -- 步進馬達驅動 |
| [9] --- 紅燈 LED      |
| [8] --- 綠燈 LED      |
| [12] -- 藍燈 LED      |
| [7] --- 蜂鳴器        |
| [6] --- 按鈕          |
| [5V] --- LCD, HC-SR04, 步進馬達, LED |
| [GND] -- 所有元件接地 |
+-------------------+
```

---

### 元件說明：
1. **LCD1602 (I2C 模組)**
   - SDA 接 A4，SCL 接 A5。
2. **超音波感應器 (HC-SR04)**
   - Trig 接 10，Echo 接 11。
3. **步進馬達 (ULN2003 驅動模組)**
   - IN1-IN4 分別接 2, 3, 4, 5。
4. **LED燈**
   - 紅燈接 9，綠燈接 8，藍燈接 12。
5. **蜂鳴器**
   - 接 7。
6. **按鈕**
   - 接 6，並加 10kΩ 上拉電阻。

---

完成接線後，請參考 `circuit_wiring.md` 獲取更詳細的接線說明，並將程式碼上傳至 Arduino 測試互動效果！
